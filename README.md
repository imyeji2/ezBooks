## ✔️ 기획의도 및 목표
   - JAVA와 DB를 최대한 활용하여 서점의 비지니스 프로세스를 관리하는 ERP 시스템을 구현
   - 상품 계약, 사원관리, 거래처 관리, 물류, 판매 등 기업 내부의 전반적인 업무를 관리
   - 상품의 판매 기록을 기반으로 판매 통계를 재공하여 기업의 상품 개발 및 물류를 효율적으로 관리
